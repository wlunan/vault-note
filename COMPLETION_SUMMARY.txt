🎯 核心模块说明
─────────────────────────────────────────────────────────────────

1️⃣  加密工具 (src/utils/crypto.ts)
   • hashNoteName()        - SHA-256 笔记 ID
   • deriveKey()          - AES-GCM 密钥派生
   • encryptNote()        - 加密内容 + 认证
   • decryptNote()        - 解密内容
   • verifyPassword()     - 验证密码正确性

2️⃣  数据库客户端 (src/lib/supabase.ts)
   • fetchNote()          - 查询笔记
   • saveNote()           - 保存/更新笔记
   • NoteRecord 接口      - 数据类型定义

3️⃣  主界面组件 (src/components/NoteEditor.vue)
   • 登录界面             - 输入名称和密码
   • 编辑界面             - 大型 textarea
   • 自动保存             - 1 秒防抖延迟
   • 状态管理             - 完整的响应式状态
   • 错误处理             - 用户友好提示

🚀 快速启动
─────────────────────────────────────────────────────────────────

1️⃣  安装依赖
    npm install

2️⃣  配置 Supabase
    编辑 .env.local，填入：
    - VITE_SUPABASE_URL
    - VITE_SUPABASE_ANON_KEY

3️⃣  创建数据库表
    在 Supabase SQL Editor 执行：
    supabase/migrations/20260222000000_create_notes_table.sql

4️⃣  启动开发服务器
    npm run dev

5️⃣  打开浏览器
    访问 http://localhost:5173
