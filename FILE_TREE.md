─────────────────────────────────────────────────────────────────
✅ 端到端加密            Web Crypto API (AES-256-GCM)
✅ 密钥派生             SHA-256(noteName + password)
✅ 用户认证            固定字符串加密验证
✅ 自动保存            1 秒防抖延迟
✅ 密码保护            无明文存储
✅ RLS 策略            所有操作允许（数据已加密）
